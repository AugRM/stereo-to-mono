# Stereo to Mono
Simple batch stereo to mono converter for WAV and MP3 files.

**For `.mp3` files [ffmpeg](https://ffmpeg.org) is necessary.**

Place this program into a folder and the script will run through all the files in the folder, find the `.wav` files and convert them to mono, creating a copy with the [MONO] label.

# Conversor Estéreo para Mono
Programa simples para conversão em massa de arquivos WAV estéreo para mono.

Coloque esse programa em uma pasta e o _script_ rodará pela pasta buscando arquivos `.wav` e convertendo-os para mono, criando uma cópia e colocando a etiqueta [MONO] no título.
